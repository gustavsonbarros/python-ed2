# 📁 Projeto Python ED2 — Estruturas de Dados

Este repositório contém testes e pequenos projetos desenvolvidos com foco em **estruturas de dados e algoritmos de busca**, pensados para quem está aprendendo e quer ver as coisas funcionando na prática.

## 🔍 O que tem aqui?

### 📌 Comparações de algoritmos de busca
- **Busca Linear**
- **Busca Binária**
- **Busca com Hashing (dicionários)**
- Comparação de desempenho entre elas com gráfico usando `matplotlib`

### 🧪 Mini projetinho prático com hashing
- **Validador de dados de usuários**
- Simula um pequeno sistema com cadastro, verificação, remoção e listagem usando dicionários como estrutura de hash.

## 🎓 Objetivo

Explorar e aplicar conceitos de estruturas de dados de forma simples e prática, com exemplos que um programador iniciante ou estagiário provavelmente usaria no dia a dia. O foco é entender na prática o **como** e o **porquê** de cada algoritmo.
