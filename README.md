Mas primeiro temos que entender, quais os fundamentos de um Hash.

As funções de hash são ferramentas fundamentais na ciência da computação, utilizadas para transformar dados de entrada (de tamanho arbitrário) em uma saída de tamanho fixo, geralmente representada por uma sequência de caracteres ou números. Essa saída é conhecida como valor hash ou digest.

Os fundamentos das funções de hash se baseiam em três principais características:

- **Determinismo:** a mesma entrada sempre gera a mesma saída.

- **Eficiência:** a função deve ser rápida de calcular.

- **Unicidade:** deve ser difícil encontrar duas entradas diferentes que produzam o mesmo hash (colisão).

As funções de hash têm diversas aplicações, incluindo:

- Verificação de integridade de dados;

- Armazenamento de senhas de forma segura;

- Estruturas de dados como tabelas hash;

- Algoritmos de criptografia e blockchain.
